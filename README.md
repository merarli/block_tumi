# ルール
基本的なルールは囲碁と同じで5個以上縦横に並べたほうが勝利

# 実行方法(run)

このプロジェクトをクローンした後に、プロジェクトルートに移動後、以下を実行してください

```
$ npm install
```

```
$ npm run dev
```
