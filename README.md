# ルール
基本的なルールは囲碁と同じで5個以上縦横に並べたほうが勝利
囲碁と違う点として、
・必ず設置するブロックは必ず地面に設置しないといけない
・3人以上でも遊べる

# 実行方法(run)

このプロジェクトをクローンした後に、プロジェクトルートに移動後、以下を実行してください

```
$ npm install
```

```
$ npm run dev
```
